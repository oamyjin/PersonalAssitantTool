# Focus Trace 使用指南 🕐

> 灵感来自桦泽紫苑《精英都是时间控》（神・時間術）：
> **想管理时间，先看清时间去哪了；把最重要的事，放进专注力最高的时段。**

一个文件、零安装、数据全在本地。打开 `focus_trace.html` 就能用。

---

## 30 秒上手

1. 用浏览器打开 `focus_trace.html`（建议 Chrome，可以记住导出位置）
2. 做完一件事，填上 **From / To / Event**，点一下专注度 1–5，**Add daily item** — 完事
3. 就这样。记录一条 ≈ 10 秒，比发一条微信还快

> 记录本身不能花时间，否则记录就成了新的时间黑洞。所以整个工具只围绕"少点几下"设计。

---

## 每周一早上 · 5 分钟开周会（和自己）

打开 **Week setup** 面板：

| 做什么 | 在哪做 |
|---|---|
| 列出本周要完成的事 | 按 **Research**（科研）/ **Personal**（生活）/ **Fun**（玩耍）三栏添加 |
| 标注几号有 meeting、约饭、deadline | **Week events** 里选日期 + 时间（可不填）+ 事件 |
| 定锚点 | **Week start** 选周一，Save week setup |

Week events 会自动按天分组显示，当天的事件也会出现在 Daily Trace 顶部的 **Schedule** 里，不会忘。

## 每天早上 · 2 分钟列清单

- **Daily todos**：把今天的具体事项拆出来，比如 `Research / 项目A design`、`Research / 跑实验`，越具体越好勾掉
- **Schedule**：今天几点开会、几点和谁见面，填时间 + 描述即可

到点了？Schedule 里每条事件有个 **Log** 按钮 — 点一下，开始时间和事件名自动填进记录表单，结束后补个 To 时间就入账了。

## 每天随手 · 记录时间流向

- 干完一件事记一条：From / To / Event / 专注度（1–5）
- 生活类的事（吃饭、休息）勾 **Personal time**，不用打专注分
- 事情还没结束？To 留空或填 `-`，之后再补
- 记录会自动关联当天的 todo（也可以手动选）

**专注度怎么打？** 5 = 心流，忘记时间；3 = 正常干活；1 = 人在工位魂在天边。打分不用纠结，凭直觉，趋势比精度重要。

## 每天睡醒 · 10 秒打卡

**Daily check-in** 面板：

- 昨晚几点躺下（In bed）、今早几点睁眼（Woke up）→ 自动算睡眠时长，对照 **8h 目标**
- 今天动了没？勾 **Exercised** — 偷懒福利：如果你记录里有 badminton / gym / 跑步 / 健身 之类的词，会**自动帮你勾上** 🏃

## 每周日晚上 · 让 AI 当你的复盘教练

1. Week setup 里点 **Copy AI prompt**
2. 粘贴到 Claude / ChatGPT
3. 它会拿到你这周的计划、事件、每一条时间记录、睡眠和运动数据，然后告诉你：
   - 时间实际去哪了（科研 / 生活 / 玩耍占比 vs 你的计划）
   - 你的专注高峰在几点，最重要的事有没有放在那里
   - 本周最大的时间黑洞 + 一个具体修法
   - 睡眠 8h、每周运动 3+ 次达标了吗
   - 下周的 3 条小规则（小到可执行、可打勾）

不想用 AI？**Dashboard** 里也有即时反馈：每日统计、**Focus by hour**（一眼看到你的专注高峰时段）、分类时长、**Week health**（一周睡眠条形图 + 运动打卡点）。

## 数据安全 🔒

- 所有原始数据只存在**你浏览器的 localStorage** 里，不联网、无服务器、没有账号
- **Download XLSX**：一键导出全部历史（Sessions / Week Setup / Daily Todos / Events / Check-ins 五张表），Chrome 里第一次选好位置后以后一键覆盖
- **Export JSON**：完整备份；**Import backup** 可导回 XLSX 或 JSON
- 换电脑 = 导出 → 拷走 → 导入，就这么朴素

**唯一的坑**：浏览器"清除浏览数据"会把 localStorage 一起清掉。养成习惯：**每周复盘时顺手点一次 Download XLSX**。

---

## 心法（来自原书的三个提醒）

1. **先记录，再优化。** 前两周什么都不用改，只记录。你对自己时间的直觉大概率是错的，数据会让你吃一惊
2. **专注高峰是稀缺资源。** 找到你的绿色时段（Focus by hour 越绿越专注），把科研里最难的事锁进去；会议、邮件、杂事丢到低谷时段
3. **睡眠和运动不是休息，是生产力。** 睡够 8 小时的一天 > 熬夜硬撑的一天半；运动后的一小时是免费的专注力充值
